Homework Ordering System respects your privacy.

Information Collected:

This extension stores the following information locally on your device:

Subjects
Teachers
Homework tasks
Task priorities
User preference settings (such as weighting values)

This information is stored using Chrome's local storage (chrome.storage.local) and is not transmitted to any external servers.

Website Access

The extension accesses the St Michael's Learning Portal timetable page solely to read timetable information needed to create and organise homework tasks.

The extension does not modify, collect, or transmit any personal information from the website beyond what is required for its functionality.

Data Sharing

This extension:

Does not collect personal information.
Does not sell or share user data.
Does not transmit user data to third parties.
Does not use analytics or advertising services.
Data Security

All extension data remains on the user's device unless the user chooses to export or share it manually.

Changes

This privacy policy may be updated if the extension's functionality changes.
